# monedadelapatria
Funcionalidades implementadas:
1. Sistema de Reportes JSON
Generación de reportes: Se crea un reporte completo en formato JSON con todas las transacciones

Organización por usuario: Las transacciones se agrupan por cada usuario

Resumen general: Incluye estadísticas generales del sistema

Descarga de archivo: Permite descargar el reporte como archivo JSON

2. Autenticación para Compras
Contraseña inicial: "12345" (almacenada en localStorage)

Modal de autenticación: Se muestra al intentar realizar una compra

Cambio de contraseña: Los usuarios pueden cambiar la contraseña del sistema

Validación: Solo se permite realizar compras después de autenticarse correctamente

3. Mejoras adicionales
Registro de transacciones: Se guarda un historial completo de todas las operaciones

Persistencia: Los datos se almacenan en localStorage para mantenerlos entre sesiones

Interfaz mejorada: Botones adicionales para generar reportes y cambiar contraseña

El sistema ahora es más seguro y proporciona herramientas completas para el seguimiento de las operaciones realizadas.
